# Feedback

Made by Dan O'Leary for friends, builders, and people tired of generic AI UI.

The best feedback is a real repo, a real `DESIGN.md`, and one place where Codex still drifted.

## Best Paths

- Open an issue for install problems, confusing docs, broken links, or skill behavior bugs.
- Open a pull request for concrete fixes: better wording, stronger examples, clearer DESIGN.md rules, or compatibility notes.
- Share a before/after screenshot when the skill works or fails in an interesting way.
- Start a broader conversation on X, LinkedIn, OpenAI Forum, or the OpenAI Developers Discord when the question is more about agent design practice than this repo.

## Great PRs Include

- What Codex did before.
- What `DESIGN.md` said.
- What changed.
- How you tested it.

Small, useful, and proven beats big and vague.

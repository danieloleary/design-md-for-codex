# Security

Made by Dan O'Leary for friends, builders, and people tired of generic AI UI.

Do not post secrets in public issues.

This repo is a Codex skill and static documentation. It should not need credentials. If you find a security concern, use GitHub's private vulnerability reporting for this repo when available. If that is not available, open a minimal public issue that says you have a private security report, without posting the details.

When adapting the skill, keep private tokens, internal URLs, customer data, and proprietary brand assets out of public `DESIGN.md` examples.
